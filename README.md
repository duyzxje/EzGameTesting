# Pages & Co. - Bookstore

Một cửa hàng sách trực tuyến nghệ thuật được xây dựng bằng HTML, Vanilla CSS, JavaScript và đóng gói bởi Vite.

## Demo trực tuyến
Trang web đã được deploy tại: [https://ez-game-testing.vercel.app/](https://ez-game-testing.vercel.app/)

## Hướng dẫn chạy ở Local

### 1. Cài đặt dependencies
```bash
npm install
```

### 2. Chạy chế độ Development
```bash
npm run dev
```

### 3. Build Production
```bash
npm run build
```
